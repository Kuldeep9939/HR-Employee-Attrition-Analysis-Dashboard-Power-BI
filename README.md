# HR Employee Attrition Analysis Dashboard

## Project Overview
This project analyzes employee attrition using the IBM HR Analytics dataset.
The dashboard helps HR teams understand attrition patterns and identify key retention drivers.

## Dashboard KPIs
- Total Employees: 1470
- Employees Left: 237
- Employees Active: 1233
- Average Tenure: 7.01 years
